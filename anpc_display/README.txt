Automatically displays the mandatory SAL and SOL links and icons for online stores in Romania.

== Description ==

This plugin automatically adds the two mandatory icons for online merchants in Romania to the website footer, according to ANPC Order no. 449/2022:
1. SAL (Alternative Dispute Resolution)
2. SOL (Online Dispute Resolution)

It is a lightweight solution with no external dependencies (images are stored locally to respect privacy/GDPR).

**Features:**
*   **Plug & Play:** Works immediately upon activation.
*   **Privacy First:** Local assets (no hotlinking).
*   **Configurable:** Customize SAL/SOL links, display position, and icon size from admin panel.
*   **Responsive:** Icons scale correctly on mobile devices.
*   **Display Position:** Choose between footer or header placement.
*   **Icon Sizing:** Three size presets (Small, Medium, Large).
*   **Custom CSS:** Add your own styling without editing files.
*   **New Tab Control:** Choose whether links open in new tabs.
*   **Live Preview:** See changes in admin panel before saving.

== Installation ==

1. Upload the `anpc_display` folder to `oc-content/plugins/` or install via Osclass admin.
2. Go to **Plugins** and click **Install** next to "ANPC Display".
3. Check your footer to see the icons.
4. Settings can be found in the admin menu under "ANPC Display".
