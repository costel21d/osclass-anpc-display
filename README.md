# ANPC Display Plugin for OSClass

Plugin OSClass pentru afișarea linkurilor obligatorii ANPC SAL și SOL în footer.

## Funcționalitate
- afișează link ANPC SAL
- afișează link ANPC SOL
- configurare din admin
- injectare automată în footer
- fără dependențe externe

## Instalare
1. Descarcă arhiva ZIP.
2. În panoul de administrare OSClass:
   Plugins → Add Plugin → Upload Plugin
3. Activează pluginul.
4. Configurează URL-urile din meniul pluginului.

Sau manual:
- copiază folderul pluginului în: oc-content/plugins/

## Compatibilitate
- OSClass 4.x
- toate temele standard
